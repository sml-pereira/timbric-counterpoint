# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-04-04

### Added
- Initial public release of the `timbric_counterpoint.pd` patch.
- Three MIDI voices (`counterpoint_1_a.mid`, `counterpoint_2_a.mid`,
  `counterpoint_3_a.mid`) transcribing Bach's Invention No. 9 in F minor.
- C4 piano sample (`yamaha_c4.wav`) used by the wavetable voice.
- MIT licence for code; CC BY 4.0 licence for data.
- README, citation metadata and repository scaffolding.
